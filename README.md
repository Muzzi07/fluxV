# fluxV (flux Five)

> **Async-First Python Library for MetaTrader 5 Algorithmic Trading**
> Unified API for backtesting and live trading.
## Features

- **Async-First**: Non-blocking operations for high-performance trading
- **Unified Interface**: Same code works in backtesting and live trading
- **Complete MT5 Wrapper**: Full access to MT5 trading capabilities
- **Backtesting Engine**: Simulate strategies with local data
- **Strategy Templates**: Ready-to-use strategies (MA Crossover, RSI)
- **Performance Analytics**: Comprehensive backtest metrics
- **Type Hints**: Full type safety with Python 3.12
